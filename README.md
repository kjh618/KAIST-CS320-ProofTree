# Proof Tree Drawer for CFWAE
A proof tree drawer for CFWAE (Conditionals, Functions, "With"'s, Arithmetic Expressions) with automatic reducing for long expressions and values.


## Usage
```console
$ cd proof-tree
$ sbt "run input_file output_file"
```
